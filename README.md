
# About HTML

- 들여쓰기는 공백 4개이거나 2개로 한다.
- Block Element를 사용할 때에는 반드시 Line Break를 넣어주자. 예를 들면다음과 같다.
    ~~~
    <p>
        I am Paragraph.
    </p>
    ~~~

* 문자 Escape 처리하기
    - `&` 는 `&amp;`로 사용한다.
    - `>` 는 `&gt;`로 사용한다.
    - `<` 는 `&lt;`로 사용한다.
    - `"` 는 `&quot;`로 사용한다.
    - `'` 는 `&#39;`로 사용한다.

- `<img>` 태그를 사용할 때에 반드시 `alt` 속성을 넣어주자.
